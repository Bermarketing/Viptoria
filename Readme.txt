# Viptoria - Landing Page para Venta de Tarjetas SIM

Este proyecto es una **landing page responsive** desarrollada para promocionar y vender tarjetas SIM de **Viptoria**. Fue creada con enfoque en la usabilidad, el diseño atractivo y la optimización de conversiones.

## ✨ Características principales

- ✅ Diseño moderno y adaptable (responsive)
- ✅ Información clara sobre los planes y beneficios
- ✅ Llamados a la acción destacados
- ✅ Secciones optimizadas para mejorar la experiencia del usuario
- ✅ Enlace a WhatsApp para contacto directo

## 🛠️ Tecnologías utilizadas

- HTML5  
- CSS3  
- JavaScript
